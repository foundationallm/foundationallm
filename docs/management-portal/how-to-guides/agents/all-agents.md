# All Agents

Learn how to view and manage all agents in the Management Portal.

## Overview

The **All Agents** view displays every agent configured in your FoundationaLLM instance.

## Accessing All Agents

1. Navigate to the Management Portal
2. Click **All Agents** in the sidebar

## Agent List

The list displays:
- Agent name
- Display name
- Type
- Status
- Created/Modified dates

## Actions

### View Agent Details
Click on an agent to view its full configuration.

### Edit Agent
Select an agent and click Edit to modify its configuration.

### Delete Agent
Select an agent and click Delete to remove it.

> **Note:** Deleted agents are soft-deleted and can be purged later.

## Filtering and Sorting

- Filter by agent type
- Sort by name, date, or status
- Search for specific agents
