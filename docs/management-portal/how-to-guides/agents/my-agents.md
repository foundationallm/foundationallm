# My Agents

Learn how to view and manage agents you own in the Management Portal.

## Overview

The **My Agents** view displays agents that you have created or have ownership of.

## Accessing My Agents

1. Navigate to the Management Portal
2. Click **My Agents** in the sidebar

## What's Included

My Agents shows agents where you are:
- The creator
- An assigned owner
- Have full management permissions

## Managing Your Agents

### Quick Actions
- Edit agent configuration
- View agent statistics
- Duplicate an agent
- Delete an agent

### Ownership Transfer
Contact your administrator to transfer agent ownership.
