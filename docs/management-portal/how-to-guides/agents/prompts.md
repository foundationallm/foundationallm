# Managing Prompts

Learn how to create and manage prompts in the Management Portal.

## Overview

Prompts are text templates that define agent behavior and instructions.

## Accessing Prompts

1. Navigate to the Management Portal
2. Click **Prompts** in the Agents section

## Creating a Prompt

1. Click **Create Prompt**
2. Configure the prompt settings:
   - **Prompt Name**: Unique identifier
   - **Description**: Purpose of the prompt
   - **Category**: Workflow or Tool
   - **Prompt Prefix**: Main prompt content
   - **Prompt Suffix**: Optional closing content
3. Click **Create Prompt**

## Prompt Categories

### Workflow Prompts
Used by agent workflows for orchestration and tool selection.

### Tool Prompts
Used by specific tools for their internal logic.

## Editing Prompts

1. Select a prompt from the list
2. Click Edit
3. Modify the configuration
4. Save changes

## Using Prompts in Agents

Prompts are referenced by agents through:
- Main workflow prompt configuration
- Tool resource assignments
- Additional workflow resources
