# API Endpoints

Learn how to manage API endpoint configurations in the Management Portal.

## Overview

API Endpoints define the connection settings for external services.

## Accessing API Endpoints

1. Navigate to **Models and Endpoints**
2. Click **API Endpoints**

## Endpoint Configuration

Configure endpoints for:
- Azure OpenAI services
- OpenAI services
- Custom model endpoints
- External APIs

## Adding an Endpoint

1. Click **Add Endpoint**
2. Select the endpoint type
3. Configure settings:
   - Endpoint URL
   - API version
   - Authentication method
   - API key or credentials
4. Save the configuration

## Authentication Types

### Key-Based
Use API keys stored in configuration.

### Token-Based
Use managed identities or OAuth tokens.

## Endpoint Health

Monitor endpoint status:
- Active: Endpoint responding
- Inactive: Connection issues
- Rate limited: Throttling in effect
