# Deployment Information

View information about your FoundationaLLM deployment.

## Overview

The Deployment Information section provides details about your current deployment.

## Accessing Deployment Information

1. Navigate to **FLLM Platform** in the sidebar
2. Click **Deployment Information**

## Available Information

### Version Information
- Platform version
- API versions
- Component versions

### Deployment Details
- Instance ID
- Deployment type (ACA/AKS)
- Region/Location

### Resource Information
- Connected Azure resources
- Service endpoints
- Configuration status

### Health Status
- Service health
- Component status
- Dependency status

## Using This Information

- Verify deployment version for support
- Check component health
- Monitor system status
