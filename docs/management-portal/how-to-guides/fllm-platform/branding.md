# Branding Configuration

Learn how to customize the look and feel of your FoundationaLLM deployment.

## Overview

Branding allows you to customize the portal appearance to match your organization's identity.

## Accessing Branding Settings

1. Navigate to **FLLM Platform** in the sidebar
2. Click **Branding**

## Customizable Elements

### Logo
- Upload your organization's logo
- Configure logo placement

### Colors
- Primary color
- Secondary color
- Accent colors

### Text
- Portal title
- Welcome messages
- Footer text

## Configuration Methods

Branding can be configured through:
1. **Management Portal** - Interactive UI
2. **App Configuration** - Azure App Configuration values
3. **REST API** - Programmatic updates

## Applying Changes

1. Make your branding changes
2. Save the configuration
3. Changes take effect after cache refresh
