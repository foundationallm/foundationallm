# Data Pipeline Runs

Learn how to view and manage data pipeline execution history.

## Overview

The Data Pipeline Runs section shows the execution history of all pipelines.

## Accessing Pipeline Runs

1. Navigate to **Data Pipeline Runs** in the sidebar
2. View the list of all pipeline executions

## Run Information

Each run displays:
- **Pipeline Name**: The pipeline that was executed
- **Status**: Current state of the run
- **Start Time**: When execution began
- **End Time**: When execution completed
- **Duration**: Total execution time

## Filtering Runs

Filter the list by:
- Pipeline name
- Status
- Date range

## Run Details

Click on a run to view:
- Complete step-by-step execution log
- Items processed in each step
- Error details (if any)
- Performance metrics

## Managing Runs

### Cancel Running Pipelines
Stop an in-progress pipeline if needed.

### Retry Failed Runs
Re-execute a failed pipeline run.
