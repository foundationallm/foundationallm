# Creating Your First Agent (Management Portal)

A quick guide to creating your first agent using the Management Portal.

## Prerequisites

- Access to the Management Portal
- Appropriate permissions to create agents
- Understanding of your agent's purpose

## Quick Steps

### 1. Navigate to Create New Agent
Click **Create New Agent** in the sidebar navigation.

### 2. Configure Basic Settings
- **Agent Name**: Unique identifier for the agent
- **Display Name**: User-friendly name shown in the portal
- **Description**: Brief description of the agent's purpose

### 3. Configure Agent Type
Select the appropriate agent type (Knowledge Management is currently supported).

### 4. Set Up Knowledge Source
Configure how the agent accesses information:
- Select data sources
- Configure indexing profiles

### 5. Configure Behavior
- Enable/disable conversation history
- Configure gatekeeper settings
- Set system prompt

### 6. Add Tools (Optional)
Configure tools like Code Interpreter or Knowledge Search.

### 7. Save and Test
- Click **Create Agent**
- Test in the Chat User Portal
