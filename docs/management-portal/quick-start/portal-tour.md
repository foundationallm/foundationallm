# Tour of the Management Portal

A guided tour of the FoundationaLLM Management Portal interface.

## Overview

The Management Portal is the administrative interface for configuring and managing your FoundationaLLM deployment.

## Main Navigation

The portal includes the following main sections:

### Agents
- **Create New Agent**: Create and configure new agents
- **All Agents**: View and manage all agents
- **My Agents**: View agents you've created or own
- **Prompts**: Manage prompt templates

### Data
- **Data Sources**: Configure data source connections
- **Data Pipelines**: Create and manage data pipelines
- **Data Pipeline Runs**: Monitor pipeline execution
- **Knowledge Sources**: Configure knowledge bases

### Models and Endpoints
- **AI Models**: Manage AI model configurations
- **API Endpoints**: Configure API endpoints

### Security
- **Instance Access Control**: Manage access permissions

### FLLM Platform
- **Branding**: Customize portal appearance
- **Configuration**: Platform settings
- **Deployment Information**: View deployment details

## Getting Started

1. Log in with your credentials
2. Navigate using the sidebar menu
3. Select the section you need to configure
