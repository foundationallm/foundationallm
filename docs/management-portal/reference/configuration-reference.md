# Configuration Reference

Reference documentation for FoundationaLLM configuration settings.

## Overview

This document provides a reference for all configurable settings in FoundationaLLM.

> **Note:** This reference is generated from the source code configuration constants.

## Configuration Sources

Settings are organized across:
- Azure App Configuration
- Azure Key Vault (secrets)
- Environment variables

## Configuration Categories

### Core API Settings
Settings for the Core API service.

### Management API Settings
Settings for the Management API service.

### Gateway API Settings
Settings for the Gateway service.

### Orchestration Settings
Settings for LangChain and Semantic Kernel orchestrators.

### Storage Settings
Settings for Azure Storage connections.

### Authentication Settings
Settings for Entra ID authentication.

## App Configuration Key Reference

See the deployed App Configuration resource for the complete list of keys and their current values.

## Environment Variables

Common environment variables:
- `ASPNETCORE_ENVIRONMENT`
- `AZURE_CLIENT_ID`
- `AZURE_TENANT_ID`
