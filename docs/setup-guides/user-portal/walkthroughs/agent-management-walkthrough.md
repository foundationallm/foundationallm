# Agent Management Walkthrough

This step-by-step walkthrough guides you through managing your personal agent catalog in the User Portal.

## Prerequisites

- Access to the FoundationaLLM User Portal
- At least one agent available in your instance

## Steps

### Step 1: Open the User Portal

Navigate to your organization's User Portal URL and log in with your credentials.

<!-- [TODO: Add screenshot of User Portal login page] -->

### Step 2: Access Settings

1. Look for the **Settings** icon (⚙️) in the top navigation bar
2. Click the Settings icon to open the Settings dialog

<!-- [TODO: Add screenshot highlighting Settings icon location] -->

### Step 3: Navigate to Agents Tab

1. In the Settings dialog, click on the **Agents** tab
2. You'll see a list of all agents available to you

<!-- [TODO: Add screenshot of Settings dialog with Agents tab selected] -->

### Step 4: Review Available Agents

The agent list displays:
- Agent name and description
- Current enabled/disabled status
- Featured agent indicators

<!-- [TODO: Add screenshot of agent list with labels] -->

### Step 5: Enable or Disable Agents

To customize which agents appear in your dropdown:

1. Find the agent you want to modify
2. Click the **toggle switch** next to the agent name
3. Toggle ON to show in your dropdown, OFF to hide

<!-- [TODO: Add screenshot showing toggle interaction] -->

### Step 6: Use the Search Filter

To quickly find a specific agent:

1. Click in the **Search** field at the top of the agent list
2. Type part of the agent name
3. The list filters in real-time as you type

<!-- [TODO: Add screenshot of search filter in use] -->

### Step 7: Filter by Status

To see only enabled or featured agents:

1. Use the **Show** dropdown menu
2. Select from:
   - **All Agents**
   - **Enabled Only**
   - **Featured Only**

<!-- [TODO: Add screenshot of filter dropdown] -->

### Step 8: Set a Default Agent

To set an agent as your default:

1. Find the agent you want as default
2. Click the **Star** icon (☆) next to the agent
3. The star fills in (★) to indicate default status

<!-- [TODO: Add screenshot showing default agent selection] -->

### Step 9: Save Your Changes

1. Review your selections
2. Click the **Save** button at the bottom of the dialog
3. Wait for the confirmation message

<!-- [TODO: Add screenshot of Save button and confirmation] -->

### Step 10: Verify Changes

1. Close the Settings dialog
2. Click on the **Agent dropdown** in the chat interface
3. Verify that only your enabled agents appear

<!-- [TODO: Add screenshot of updated agent dropdown] -->

## Result

After completing this walkthrough, you have:

- ✅ Accessed the agent settings
- ✅ Enabled/disabled agents based on your preferences
- ✅ Used filters to find specific agents
- ✅ Set a default agent (optional)
- ✅ Saved and verified your changes

## Tips

- **Can't find an agent?** You may not have permission to access it. Contact your administrator.
- **Agent disappeared?** It may have been deleted or expired. Check with the agent owner.
- **Changes not saving?** Try refreshing the page and signing in again.
