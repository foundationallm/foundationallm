# Copying Prompts & Formatted Results

Learn how to copy prompts and results from your conversations.

## Overview

The Chat User Portal provides copy functionality that preserves formatting for use in other applications.

## Dual-Format "Copy Message" Behavior

When copying messages, the system may provide:
- **Markdown format**: Raw markdown for use in compatible editors
- **Rich Text format**: Formatted text for word processors and emails

## Copying Messages

1. Hover over or click on a message
2. Look for the copy icon or menu
3. Click to copy the content
4. Paste into your target application

## Expected Formatting Output

### Markdown Output
- Code blocks preserved with syntax
- Lists and headers maintained
- Links formatted as markdown

### Rich Text Output
- Formatted text with styling
- Tables rendered properly
- Code displayed in monospace font

## Limitations

- Some complex formatting may not transfer perfectly
- Large messages may be truncated
- Special characters may need escaping
