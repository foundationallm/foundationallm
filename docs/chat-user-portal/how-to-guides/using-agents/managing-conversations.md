# Managing Conversations

Learn how to create, view, and manage your conversations in the Chat User Portal.

## Overview

Conversations in FoundationaLLM are sessions where you interact with an agent. Each conversation maintains its own history and context.

## Creating a New Conversation

1. Click the **New Conversation** or **+** button
2. Select an agent (if not already selected)
3. Start typing your first message

## Viewing Conversation History

1. Look for the conversation list in the sidebar
2. Click on a previous conversation to view it
3. Continue the conversation or review past messages

## Conversation Features

### Context Retention
- Agents can remember previous messages in the conversation
- The number of messages retained is configured by the agent settings

### Session Management
- Each conversation has a unique session ID
- Sessions persist across browser sessions (when logged in)

## Deleting Conversations

1. Locate the conversation in the sidebar
2. Click the delete/trash icon
3. Confirm the deletion

> **Note:** Deleted conversations cannot be recovered.
