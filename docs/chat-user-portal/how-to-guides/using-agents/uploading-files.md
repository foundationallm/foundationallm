# Uploading Files to a Conversation

Learn how to upload files to your conversation for agent processing.

## Overview

Some agents support file uploads, allowing you to share documents, images, and other files for the agent to analyze and reference.

## Prerequisites

- The agent must have file upload enabled
- Your file must be in a supported format
- File size must be within limits

## Supported File Types

Common supported formats include:
- Documents: PDF, DOCX, DOC, TXT, MD
- Spreadsheets: XLSX, XLS, CSV
- Images: PNG, JPG, JPEG, GIF
- Code files: PY, JS, TS, and more

> **Note:** Supported file types may vary by agent configuration.

## Uploading Files

1. Look for the file upload button (ðŸ“Ž or similar icon)
2. Click to open the file picker
3. Select your file(s)
4. Wait for the upload to complete
5. The agent will acknowledge the file and can reference it

## File Processing

After upload:
- Files are processed by the agent's data pipeline
- Content is extracted and made available to the agent
- You can ask questions about the uploaded content

## File Size Limits

- Check with your administrator for specific limits
- Large files may take longer to process
