# Printing Conversations

Learn how to print your conversation history.

## Overview

Print functionality allows you to create physical or PDF copies of your conversations.

## Printing a Conversation

1. Navigate to the conversation you want to print
2. Look for the print option in the menu
3. Click to open the print dialog
4. Configure your print settings
5. Print or save as PDF

## Print Options

- **Print current conversation**: Print the active conversation
- **Print selection**: Print selected messages only
- **Save as PDF**: Create a PDF file instead of printing

## Formatting

Printed conversations typically include:
- User prompts and agent responses
- Timestamps
- Agent identification
- Basic formatting preserved
