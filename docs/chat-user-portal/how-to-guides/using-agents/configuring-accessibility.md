# Configuring Accessibility

Learn about accessibility features in the Chat User Portal.

## Overview

The Chat User Portal includes accessibility features to ensure all users can effectively interact with AI agents.

## Accessibility Features

### Keyboard Navigation
- Navigate through the interface using keyboard shortcuts
- Tab through interactive elements
- Use Enter to submit messages

### Screen Reader Support
- Interface elements include ARIA labels
- Messages are announced to screen readers
- Status updates are communicated accessibly

### Visual Accessibility
- High contrast support
- Resizable text
- Focus indicators for keyboard navigation

## Browser Settings

For the best accessibility experience:
- Enable your browser's accessibility features
- Configure your preferred screen reader
- Adjust browser zoom levels as needed
