# Selecting an Agent

Learn how to select and switch between available agents in the Chat User Portal.

## Overview

The Chat User Portal allows you to choose from multiple agents, each configured for specific tasks or knowledge domains.

## Selecting an Agent

1. Navigate to the Chat User Portal
2. Look for the agent selector dropdown in the interface
3. Click the dropdown to see available agents
4. Select the agent you want to interact with

## Agent Information

When selecting an agent, you may see:
- **Agent Name**: The display name of the agent
- **Description**: A brief description of the agent's capabilities
- **Welcome Message**: Initial message explaining what the agent can help with

## Switching Between Agents

You can switch between agents at any time:
1. Click the agent selector dropdown
2. Choose a different agent
3. A new conversation will start with the selected agent

> **Note:** Switching agents starts a new conversation. Your previous conversation with the other agent remains accessible in your conversation history.
