# Rating Agent Responses

Learn how to provide feedback on agent responses.

## Overview

Rating responses helps improve agent quality by providing feedback to administrators.

## How to Rate Responses

If enabled by the agent:
1. Look for rating controls near each response (thumbs up/down, stars, etc.)
2. Click to rate the response quality
3. Optionally provide additional feedback

## Rating Options

Common rating options include:
- **Thumbs up/down**: Simple positive/negative feedback
- **Star ratings**: Graduated quality assessment
- **Text feedback**: Detailed comments

## Why Rating Matters

Your ratings help:
- Identify areas for agent improvement
- Track agent performance over time
- Guide prompt and configuration adjustments
