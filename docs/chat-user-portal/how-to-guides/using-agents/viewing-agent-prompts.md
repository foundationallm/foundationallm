# Viewing Agent Prompts

Learn how to view the prompts used by agents in your conversations.

## Overview

If enabled, you can view the system prompts and context that agents use to generate responses.

## Prerequisites

- The agent must have prompt viewing enabled
- This feature is configured by administrators

## Viewing Prompts

1. Look for the "View Prompt" option near agent responses
2. Click to expand the prompt details
3. View the system prompt, context, and instructions

## Prompt Components

You may see:
- **System Prompt**: The agent's base instructions and persona
- **Context**: Retrieved information used for the response
- **Conversation History**: Previous messages included for context

## Why View Prompts?

- Understand how the agent generates responses
- Debug unexpected responses
- Learn about agent configuration
- Verify context retrieval
