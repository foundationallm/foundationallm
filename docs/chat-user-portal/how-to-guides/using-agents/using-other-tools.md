# Using Other Tools

Learn about additional tools available in FoundationaLLM agents.

## Overview

Beyond Code Interpreter and Knowledge tools, agents may have access to various other tools:

## Available Tools

### DALL-E Image Generation
- Generate images from text descriptions
- Create visualizations and graphics

### Custom Tools
- Organization-specific tools configured by administrators
- External API integrations

## Using Tools

Tools are invoked automatically by the agent based on your request:
1. Describe what you need
2. The agent selects the appropriate tool
3. Results are returned in the conversation

## Discovering Available Tools

To see what tools an agent has:
- Check the agent's description
- Ask the agent about its capabilities
- View agent configuration (if you have access)
