# Managing Available Agents

Learn how to view and manage the agents available to you in the Chat User Portal.

## Overview

The agents available to you in the Chat User Portal are determined by:
- Agents published to your organization
- Your role-based access permissions
- Agent visibility settings configured by administrators

## Viewing Available Agents

1. Open the Chat User Portal
2. Click the agent selector dropdown
3. View the list of agents you have access to

## Agent Visibility

Agents may be:
- **Public**: Available to all authenticated users
- **Private**: Limited to specific users or groups
- **Role-restricted**: Available based on your assigned roles

## Requesting Access to Additional Agents

If you need access to an agent not currently available to you:
1. Contact your FoundationaLLM administrator
2. Request the appropriate role assignment
3. Once granted, the agent will appear in your agent selector
